Aqui temos imagens num site:

Geralmente, as imagens contam com 3 fontes (source):

  1. Imagem que vem da pasta local de seu computador
  2. Imagem que vem de uma subpasta local de seu computador
  3. Imagem que vem de uma url direto da internet

Importante: uma imagem que vem diretamente de uma url pode cair se por acaso o site original cair.
