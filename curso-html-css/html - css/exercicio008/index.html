Principais Formatações

Negrito / Destaque

Nesta frase temos um termo em negrito ao usar a tag <b> (não semantico)

Nesta frase temos um termo em destaque usando a tag <strong>. A TAG strong é semantica, ou seja, tem significado e pode alterar resultados em sites de busca, como o Google, por exemplo.

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag <i> (não semantico)

Nesta frase, temos um termo com ênfase, usando a tag <en> (semantico)

Texto Marcado

Para criar um texto marcado usa-se a tag <mark>

Texto Pequeno

Utiliza-se a tag <small> para criar um texto pequeno, semantico. Por exemplo, letras pequenas em um contrato.

Texto Deletado

Também semantico, usa-se a tag <del> para riscar um texto, ou dar a impressão que ele foi deletado

Texto Inserido

De forma similar, usa-se a tag <ins> para um texto inserido depois

Texto Sublinhado

Não semantico, tem-se o <u> para sublinhar um texto

Texto sobrescrito

Para inserir coisas do tipo x20-3
